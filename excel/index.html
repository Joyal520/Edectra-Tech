Lesson 1

Introduction to Microsoft Excel

Objective: Understand what Excel is, navigate its interface, and identify core components to build confidence.

What is Excel?

Microsoft Excel is a powerful electronic spreadsheet program used for storing, organizing, and manipulating data. Think of it as a massive digital piece of graph paper where you can make lists, organize budgets, and perform automatic calculations.

It is used globally in homes for personal budgeting, in schools for grade tracking, and in every major office for financial health assessment, data analysis, and professional reporting.

Expand for a deep dive into the Interface
When you open Excel, you'll see a grid. Above this grid is the Ribbon—a menu system grouping commands logically (like 'Home' for fonts, 'Insert' for charts). Below the Ribbon sits the Formula Bar, a long white box showing exactly what is inside the cell you have clicked on. At the very bottom are Sheet Tabs, allowing you to have multiple pages in one file.

Key Concepts

Workbook

The entire Excel file itself (e.g., Data.xlsx). Like a physical book.

Worksheet

A single page inside the workbook. You can add many sheets.

Column

Vertical lines of data. Labelled with letters (A, B, C...)

Row

Horizontal lines of data. Labelled with numbers (1, 2, 3...)

Cell

The single box where a row and column meet.

Cell Address

The cell's specific name, calculated by its Column and Row (e.g., B2).

Lesson 2

Entering and Formatting Data

Objective: Input data correctly and make your spreadsheets legible, professional, and visually appealing using formatting tools.

Data Entry Basics

To enter data, simply click on a cell and start typing. Excel accepts two main forms of data:

  • Text (Labels): Words, names, or addresses. Excel aligns text to the left of the cell by default.
  • Numbers (Values): Numbers, dates, or prices. Excel aligns numbers to the right of the cell by default.

If you make a mistake, you can double-click the cell, or select it and press F2, or click inside the Formula Bar to edit the contents.

Formatting: Making it Look Good

Why does formatting matter? A spreadsheet is useless if others cannot read it. Using the 'Home' tab on the Ribbon, you can change aesthetics:

Bold & Size

Used to make headings stand out. Size dictates readability.

Alignment

Moving text Left, Center, or Right within a cell.

Fill Colour

Adds background colour to a cell to differentiate headers or totals.

Borders

By default, Excel gridlines don't print. Borders draw solid lines around cells.

Lesson 3

Basic Calculations in Excel

Objective: Transform Excel from a digital table into a powerful calculator using basic mathematical formulas and cell references.

What is a Formula?

A formula is an expression that calculates the value of a cell. This is where Excel comes alive.

The Golden Rule: EVERY formula in Excel MUST begin with an equals sign (=). This tells Excel "Do math here!", rather than just typing text.

Why use Cell References instead of numbers?

You could type =10+5. The result is 15. But if your 10 changes to 20 later, you have to rewrite the formula.

Instead, if you type =A1+B1, Excel looks at whatever is inside A1 and B1. If you update the data in A1, the formula recalculates the answer automatically. This is the entire point of spreadsheets!

Arithmetic Operators

Addition +

Example: =A1+A2

Subtraction -

Example: =A1-A2

Multiplication *

Example: =A1*A2 (Asterisk)

Division /

Example: =A1/A2 (Forward slash)

⚠️ Common Mistake: Typing A1+A2 without the equals sign. Excel will literally just display the text "A1+A2". Always start with =!

Lesson 4

Important Excel Functions

Objective: Upgrade from basic math to utilizing built-in "Functions" capable of analyzing vast amounts of data in a single click.

Formula vs Function

A Formula is an equation you type yourself (e.g. =A1+A2+A3+A4+A5). If you had 100 numbers, typing that would be a nightmare.

A Function is a built-in recipe inside Excel that performs a specific calculation instantly. Using a function, you can add 100 numbers just by typing =SUM(A1:A100). The colon (:) means "through" (from A1 through A100).

Top 5 Beginner Functions

  • SUM - Adds values up. e.g. =SUM(B2:B4)
  • AVERAGE - Calculates the average (mean) of the numbers.
  • MAX - Finds the absolute Highest number in the selected range.
  • MIN - Finds the absolute Lowest number in the selected range.
  • COUNT - Counts how many cells contain numeric entries.

Creating a Simple Sales Sheet

Objective: Put everything together. Build a functional, automated template used daily in shops, schools, and offices worldwide.

Why Businesses Love Excel

A business needs to know its revenue. If you sell items, you have a Quantity (how many you sold) and a Price (cost per item). Total Revenue is simply Quantity multiplied by Price.

By creating a Sales Sheet, a business owner can just type in the daily quantities, and Excel instantly calculates individual totals and the grand total for the day. You are going to build this logic now using Formulas and Functions simultaneously.

🛠 Final Activity: Build the Automated Sheet

Your task is to program Column D to calculate the Totals automatically.

  1. Calculate each item total: Write a formula in cell D2 to calculate the Total for Pens (Multiply Quantity by Price).
  2. Do the same for Books in D3, and Bags in D4.
  3. Calculate grand total: Click cell D5 (next to Grand Total). Write a Function to add up everything in column D.
Line-by-Line Solution:

D2 (Pen Total): =B2*C2 (Result comes to 20)

D3 (Book Total): =B3*C3 (Result comes to 20)

D4 (Bag Total): =B4*C4 (Result comes to 40)


D5 (Grand Total): =SUM(D2:D4) (Result comes to 80)

Congratulations! This exact structure is how 90% of basic business invoicing works. If someone returns a Bag, you just change cell B4 to '1'. Excel will update D4 to 20, and D5 to 60 instantly!
